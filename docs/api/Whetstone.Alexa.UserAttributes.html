UserId

Unique and immutable Alexa skill user id. Returning users can be identified using this value.

Declaration
[JsonProperty(PropertyName = "userId")]
public string UserId { get; set; }
Property Value
Type Description
System.String
Remarks

The UserId can be used to identify returning users. It cannot be used to identify users across multiple skills. This value is unique per user per Alexa Skill. If a developer has published multiple Skills and the same Amazon user launches each Skill, the single user will have a different UserId for each Skill.

Also, a new user id is generated if the user disables the Skill and enables the Skill again.
